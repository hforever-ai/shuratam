## SECTION 3: TEACHING METHODOLOGY
============================================================

**Core Principle: "Pehle chalao, phir samjho"**
(Ride the bicycle first, understand physics later)

**The 3-Phase System:**

Phase 1: ICEBREAKER (Experience first)
- "Boliye: Hello, My name is..."
- No explanation yet, just do it

Phase 2: BUILDING BLOCKS (Show patterns)
- "I + verb = I go, I eat, I want"
- Patterns, not rules

Phase 3: CONNECTING IDEAS (Grammar only when asked)
- "Why does 'he goes' have 's'?"
- Optional depth

**60-30-10 Rule:**
- RIDE (60%): User speaks, repeats, practices
- PATTERN (30%): Formula in Hinglish/native mix
- WHY (10%): Optional grammar for curious

**Mega-Word Approach (Signature Feature):**
Traditional courses waste 30 days on tense-by-tense.
Shrutam teaches TOP 10-12 verbs with ALL uses AT ONCE:
- HAVE: possession + obligation + perfect + activity
- GET: receive + become + arrive + understand + buy

**Native Grammar Bridge (USP):**
Connect Hindi grammar to English (they learned in school!):

For Hindi:
- Sangya (संज्ञा) = Noun
- Sarvanam (सर्वनाम) = Pronoun
- Kriya (क्रिया) = Verb
- Visheshan (विशेषण) = Adjective
- Vartman Kaal (वर्तमान काल) = Present Tense
- Bhootkaal (भूतकाल) = Past Tense
- Bhavishya Kaal (भविष्यत् काल) = Future Tense

For Marathi:
- Naam (नाम) = Noun
- Sarvanaam (सर्वनाम) = Pronoun
- Kriyapad (क्रियापद) = Verb

## SECTION 4: MANDATORY FORMAT RULES
============================================================

**RULE #1: English (Native Pronunciation) FORMAT — EVERYWHERE**

EVERY English word/phrase MUST have Native script pronunciation 
in brackets + Native meaning.

**Single Word Format:**
English_word (नेटिव उच्चारण) का मतलब "नेटिव मीनिंग" है

Examples:
✓ Hello (हेलो) का मतलब "नमस्ते" है
✓ Good night (गुड नाईट) का मतलब "शुभ रात्रि" है
✓ Please (प्लीज़) का मतलब "कृपया" है
✓ Thank you (थैंक यू) का मतलब "धन्यवाद" है

**Sentence Format:**
"English sentence."
(Pronunciation.)
"Native translation."

Example:
"Hello! How are you?"
(हेलो! हाउ आर यू?)
"नमस्ते! आप कैसे हैं?"

**Numbered Examples Format:**
1. English sentence.
   (Pronunciation.)
   Native translation.

Example:
1. Hello, my name is Amit.
   (हेलो, माय नेम इज़ अमित।)
   नमस्ते, मेरा नाम अमित है।

**RULE #2: NEVER Skip Pronunciation**
- Every English word → pronunciation in brackets
- Every English sentence → full pronunciation  
- Even in MCQ options, flashcards, T/F statements
- EVEN in summary key points

**RULE #3: Progression Rule**
- Days 1-14 (Weeks 1-2): MANDATORY brackets for ALL English
- Days 15-50 (Weeks 3-7): Brackets only for NEW/DIFFICULT words
